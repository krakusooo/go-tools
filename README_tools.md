## Additional Tools

### ipinfo
Lookup geolocation and ASN info for an IP address.
```bash
go run ipinfo/main.go -ip 8.8.8.8
```

### sysmon
Display system resource statistics.
```bash
go run sysmon/main.go
```

### banner
TCP banner grabbing for service identification.
```bash
go run banner/main.go -host example.com -port 22
```
